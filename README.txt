How to use
1) Open index.html in a browser (Chrome/Edge recommended).
2) Click any figure to open the zoom viewer.
   - Mouse wheel: zoom
   - Click+drag: pan
   - Reset: fits image to view
   - Fullscreen: optional

How to swap images (no code edits)
Overwrite the files in assets/img/ using the same filenames:
- stack_all.png
- may14_green.png
- jul07_orange.png
- sep23_yellow.png
- nov28_lightblue.png
- jan13_pink.png
